Кнопка «Матрица Эйзенхауэра» - это одна из кнопок «нижнего меню» которая показывает страницу  «Главная-Матрица Эйзенхауэра»
Кнопка «Помодоро» - это одна из кнопок «нижнего меню» которая показывает страницу  «Главная-Помодоро»
Кнопка «Настройки» - это одна из кнопок «нижнего меню» которая показывает страницу  «Главная-Настройки»
Страница «Главная-список задач»:
Кнопка «Список задач» (9) - это одна из кнопок «нижнего меню» которая показывает страницу  «Главная-Список задач». При нажатии на нее и при входе в МП открывается именно этот список.
Списки:
- просрочено
- сегодня
- завтра
- позже
- без срока
- выполнено
должны присутствовать по умолчанию на этой странице (но какие из этих списков будут видны или не видны можно будет настроить на странице «Главная-настройки», ниже опишу)
На каждом списке должно указываться кол-во задач которые входят в этот список.
Все списки находятся в свернутом состоянии, т.е.при нажатии на любой из этих списков должен выпасть список с перечнем задач входящих в этот список.
При этом задача  в этом списке должна содержать след.информацию:

--- 122 ---
Постановка задач идентично тому как указано было на других страницах только добавляется возможность выбора блока в который попадет задача 
Страница «Главная-Помодоро»:
При нажатии в нижнем меню на кнопку по форме как помидор открывается страница на которой можно настроить таймер обратного отсчета по принципу фокусировки «помодоро»
 - 1 – это кнопка в нижнем меню
 - 2 – сам таймер обратного отсчета с кольцом окончания времени вокруг цифр времени
 - 3 – можно выбрать задачу по которой осуществляется фокусировка
 - 4 – кнопка меню настроек «Помодоро»,  в ней должны быть след.настройки:
      4.1 – «продолжительность таймера» - т.е.тут пользователь может выбрать в минутах продолжительность таймера по умолчанию
      4.2 – выбрать необходимость показания таймера в режиме блокировки экрана смартфона
      4.3 – выбрать звук завершения таймера по окончании времени таймера
 
При нажатии на кнопку «Фокус» появиться окно поиска задачи по которой будет установлен таймер
    
Это кнопка позволяет выбрать мелодию которая будет проигрываться во время работы таймера.
Рядом окно выбора мелодии, при нажатии можно прослушать звук мелодии

Это сам таймер обратного отсчета и кнопки:
 - старт (плэй)
 - на кнопке старт после нажатия появляется кнопка «пауза» (при нажатии на кнопку «пауза» под цифрами таймера должна появится запись «На паузе»)
 - и кнопка «стоп», при ее нажатии таймер просто возвращается в первоначальное положение (например 30 минут) и останавливается
Когда таймер дойдет до конца должно быть звуковое оповещение об окончании этого таймера.
